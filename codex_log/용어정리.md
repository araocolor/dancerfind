# 용어 정리

## 사용자-LLM 커뮤니케이션 기준

1. 문장 끝이 `?` 로 끝나면 질문으로 처리한다.
- 작업은 수행하지 않고, 간략한 답변만 제공한다.

2. `배포`는 `커밋 + 푸시`를 의미한다.

3. `간단설명`은 아래 형식으로 답변한다.
- 기술적 용어 금지
- 웹 사용자 관점
- 3줄의 짧은 문장

4. `다시설명`은 아래 형식으로 답변한다.
- 기술적 용어 금지
- 웹 사용자 관점
- 3줄의 짧은 문장

5. `쉽게`, `다시`는 아래 형식으로 답변한다.
- 기술적 용어 금지
- 웹 사용자 관점
- 3줄의 짧은 문장

6. `로직으로 설명`은 아래 형식으로 답변한다.
- 실제 파일명 표시
- 로직 이해를 위한 상세 설명
- 3줄 이내로 정리
